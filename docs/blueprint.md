# **App Name**: Subscription Clarity

## Core Features:

- Informational Hero Section: Engaging hero section on the landing page explaining the core value proposition.
- Problem Explanation: Clearly articulate the pain points of unmanaged subscriptions.
- Key Benefit Highlights: Showcase the main benefits of using the Subscription Clarity app in a concise, visually appealing manner.
- Call-To-Action Buttons: Prominent 'Get Started' and 'Login' buttons that navigate to the respective routes.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to inspire trust and financial responsibility.
- Background color: Light gray (#F0F0F0) for a clean and modern aesthetic.
- Accent color: Teal (#3D84A8) to draw attention to key actions and information.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, giving a modern and readable appearance.
- Responsive layout that adapts to different screen sizes, ensuring usability on both desktop and mobile devices.
- Simple, outline-style icons to visually represent the benefits of the application.
- Subtle transitions and hover effects on buttons to provide visual feedback to user interactions.
